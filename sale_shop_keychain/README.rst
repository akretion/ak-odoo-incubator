==========================
Sale Shop Keychain Account
==========================

Bind keychain accounts to shop.

Usage
=====

From the shop settings (Sales > Settings > Shop), you can bind few accounts to each shops

This is consumed by other modules (like delivery_roulier_sale_shop)
